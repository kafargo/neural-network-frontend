Featured Project: Neural Network Demo

This interactive demo is built upon the excellent work of Michael Nielsen's neural network implementation from his book on deep learning, which was later modernized by unexploredtest with Python 3 compatibility. I forked this updated foundation and built a complete full-stack application around it, creating a Flask API wrapper with WebSocket integration to enable an interactive web experience. The result is a hands-on demonstration where users can create custom neural network architectures, train them on the MNIST handwritten digit dataset, and watch real-time training progress. The system allows you to:

  • Create custom neural network architectures
  • Train networks with configurable parameters (epochs, batch size, learning rate)
  • Monitor training progress in real-time via WebSocket updates
  • Test the trained network with successful and unsuccessful predictions
Built with: Python Flask Angular WebSockets NumPy
